La versión mínima de PHP es la 5.4.*
Para instalarlo:

- Primero deberás editar el archivo conexion.php que se encuentra en la carpeta principal, estableciendo los datos de usuario correctos para realizar la conexión con la base de datos.
- Una vez realizado, deberás ejecutar el siguiente comando por consola ( con la ruta iniciada en este directorio ): php install-default.php
- Finalizado
